# Exercise 1

#### Concatenate
There are two implementations to this program, one in C (`concatenate.c`) and the other in C++ (`concatenate.cpp`). The purpose was to show the differences between a higher-level and lower-level language when manipulating file descriptors (STDIN and STDOUT). The programs both directly copy data from STDIN or a file to STDOUT using direct byte techniques.

#### Unicode Count
Again, two simple implementations, one in C (`unicodecount.c`) and one in C++ (`unicodecount.cpp`). These programs count all valid UTF-8 encoded Unicode characters input to them. The purpose of this was again to familiarize us with the difference between C and C++.
